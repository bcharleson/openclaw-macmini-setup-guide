# OpenClaw Mac Mini Setup Guide

Brandon Charleson's configuration for setting up OpenClaw agents on Mac Mini with isolated user accounts.

## Quick Start

Use the [AGENT-SETUP.md](./AGENT-SETUP.md) guide to set up an OpenClaw agent with:
- Dedicated user account
- Slack integration
- OAuth or API key authentication
- Security best practices
- Admin access configuration

## Claude Skill

This guide is available as a Claude skill! See [claude-skill.md](./claude-skill.md) for usage.

## Features

- ✅ Variable-driven setup (copy-paste friendly)
- ✅ Complete Slack OAuth configuration
- ✅ Security-focused (proper permissions, isolated config)
- ✅ Works with any LLM (recommended: Anthropic Opus 4.5)
- ✅ Launchd integration for persistence
- ✅ Admin access with symlinks
